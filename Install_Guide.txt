2. Open CMD by writing cmd in address bar in zip extracted folder
3. Run Command: python -m venv venv
4. Run Command: pip install -r requirements.txt
5. Run Command: python manage.py migrate
6. Run Command: python manage.py createsuperuser
7. Give a username, then email, then password twich -> if there is a promt put y and then enter

9. Run Command: connect oracle_username/password
10. Now Go Back To Python CMD project folder
11. Run Command: python manage.py createdb
